## 做题建议

1. **按顺序做**：从第 1 题到第 20 题，难度逐渐递增
2. **先独立完成**：尽量不查笔记，实在想不起来再看
3. **运行测试**：用 `dart run "Test question/第X题_xxx.dart"` 运行验证
4. **寻求帮助**：需要提示或检查代码时随时说
5. **查看答案**：做完后说 **"给答案"**，我会把参考答案写入文件
